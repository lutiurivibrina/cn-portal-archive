# cn-portal-archive

## 项目简介

cn-portal-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本项目不针对任何特定域名或网站，仅作为一组静态页面文件的存储与展示空间。  
页面内容均以独立 HTML 文件形式存在，可通过 GitHub Pages 或其它静态托管服务直接访问。

## 目录结构

```
cn-portal-archive/
├── index.html          # 首页 / 页面索引
├── pages/              # 存放归档的 HTML 页面文件
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 样式、脚本、图片等静态资源
│   ├── css/
│   ├── js/
│   └── images/
├── README.md           # 本文件
└── .gitignore          # Git 忽略规则
```

## 页面归档说明

- 每个 HTML 页面均为独立的归档单元，可单独访问。
- 页面内容可能包含信息展示、工具页面、示例页面等，不限于特定主题。
- 所有页面均以标准 HTML 格式编写，兼容主流浏览器。
- 页面索引（index.html）提供对归档页面的导航与概览。

## 使用方式

1. **直接浏览**：通过 GitHub Pages 或其它静态托管服务访问本仓库的 `index.html` 或 `pages/` 下的任意 HTML 文件。
2. **本地查看**：克隆仓库后，用浏览器直接打开对应的 HTML 文件即可。
3. **自行部署**：可将仓库内容部署到任何支持静态文件的 HTTP 服务器。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改归档页面。
- 欢迎通过 Issue 提交建议或报告问题。
- 不接受直接对页面内容的修改请求，除非涉及明显的错误或兼容性问题。
- 页面内容不保证时效性，以归档时的状态为准。
